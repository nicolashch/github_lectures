How to contribute  to an Open source project
	1. Watch the  project
	2. Fork --> Copy of the project's actual state (Git Hub)
		Clone from fork (in my repositories)
		~ git clone (link)
	3. Make changes and then --> ~ git push
	4. at Github --> new pull request --> create pull request
	5. Wait
	6. Stay update 
		- Clone Original project in another origin 'upstream'
			~ git remote add upstream (link)
			~ git pull upstream master
			~ git commit -am
			~ git push origin master
