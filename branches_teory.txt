BRANCHES (RAMAS)
_________

Create new branch
	~ git branch <branch-name>

Create a new branch and checkout to it in one line
	~ git checkout -b <branch-name>
	
To see branches
	~ git branch
	~ git show-branch
	~ git show-branch --all

To push branch into remote repository
	~git push origin <branch name>

Stash command allows to save changes without saving them to staging
	~ git stash # Saves changes momentarly
	~ git stash list # Shows changes saved at stash
	~ git chechkout stash <branch-name> # Send changes to a new branch
	~ git stash drop # delets stash created
