BRANCHES (RAMAS)
_________

Create new branch
	~ git branch <branch-name>

Create a new branch and checkout to it in one line
	~ git checkout -b <branch-name>
	
To see branches
	~ git branch
	~ git show-branch
	~ git show-branch --all
To see local and remote branches
	~ git branch -r #shows remote branches
	~ git branch -s # Shows local and remote branches
To push branch into remote repository
	~git push origin <branch name>
